# SyBridge × Multiplier — India Hiring Microsite

A personalized microsite for SyBridge Technologies covering why an Employer of Record is the smart way to hire in India, and why Multiplier is the right partner to do it with.

## Deploying on GitHub Pages

1. Create a new GitHub repository
2. Upload `index.html` and `README.md` to the root
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Select `main` branch, `/ (root)` folder → **Save**
6. Live at `https://yourusername.github.io/your-repo-name` within ~60 seconds
